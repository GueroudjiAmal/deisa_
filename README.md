# PDI Deisa Plugin

### Installation 

Check https://github.com/pdidev/spack#deisa for spack installation.

Or it can be  installed on top of pdi with :

```
  cmake -DCMAKE_INSTALL_PREFIX=$HOME/local/  -DPython3_EXECUTABLE=~/.conda/envs/yourenv/bin/python3.8  ../
  make install
```
